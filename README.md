# 상공회의소 서울기술교육센터 인텔교육 2기

## Clone code 

```shell
git clone --recurse-submodules https://github.com/kccistc/intel-02.git
```

* `--recurse-submodules` option 없이 clone 한 경우, 아래를 통해 submodule update

```shell
git submodule update --init --recursive
```
